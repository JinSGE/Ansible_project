# 3-2_Ansible_Project
------------------------
설명
------------------------
Ansible을 이용해 DNS, FTP, 웹, DB, 메일, NTP 서버를 자동으로 구축하는 프로젝트.

서버는 아래와 같이 분산하여 구성:

Server1: DNS + FTP

Server2: WEB + DB

Server3: MAIL + NTP

각 서버에 필요한 서비스를 자동으로 구축하는 과정은 Ansible을 사용하여 실행.
